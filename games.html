About

Joompy is a FPS/Platformer where you play as an African American cyberagent Whistler hired by an Orthodox institution to
destroy a Danish empire that has taken over the world. This game was originally made in thirty minutes,
however the game has received updates that have added vast amounts of content. This game was made on a fan-made close sourced Half-Life 2 Beta engine as a
test to see how software limitations could affect gameplay, as I could not make any source code modifications.

You can download the game for free off MODDB, you must install the 'Mega-Build' from hl2-beta.ru and install it correctly.

Cat Bread Factory is a hyper-palatable clicker game where you hire felines to manufacture bread.
You can hire a variety of felines, from lowly peasants to dictators.
This game was originally made in a day.

You can purchase the game or download the demo on itch.io
